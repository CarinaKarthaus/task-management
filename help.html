JOIN - The Task Management System

What is JOIN?

JOIN is a smart tool to organize and priorize your to-dos easily.
It's based on the Eisenhower-principle which categorizes tasks based on both their importance and urgency.

How it works

  1. Create a new task
    2.
  2. Check pending tasks in Matrix view
    3.
  3. Change task urgency / importance
    4.
  4. View and edit task list
    5.

Tap on at the top-right corner to open the Navigation Menu


  1. Create a new task

    Fill out the form with all the required details.
    No worries: you can change the info later if needed.

  2. Check pending tasks in Matrix-view

    Check how and when to handle your tasks best.
    The tool will automatically assign your task to the correct category based on the provided details.

    The classification works as follows:
    Category Importance Urgency
    DO High High
    (or due date in < 7 days)
    SCHEDULE High Low
    (or due date in ≥ 7 days)
    DELEGATE Low High
    (or due date in < 7 days)
    ELIMINATE Low Low
    (or due date in ≥ 7 days)
  3. Change task urgency / importance

    Of course you can change urgency and/or importance manually if needed.
    Just select a task and shift it to a new box by pressing the arrows.

  4. View and edit task list

    Find a list with all your tasks.

    Edit your task details by clicking on the text area.
    Delete tasks by pressing the trash button.

HAVE FUN exploring JOIN!
